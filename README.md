LJMD-OpenCL
===========

Repository for ICTP School-Lab. Collaborative development of OpenCL version of LJMD by Axel K.  

Install
======

Download Sources
----------------

	$ git clone https://github.com/LJMD-OpenCL/ljmd-opencl.git

Install
-------
	make
You will receive a executable called ljmd_CL

Test
----
	make test
